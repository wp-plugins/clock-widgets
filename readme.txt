Display current date and time.

== Description ==

This plugin includes the option to display a current date and time in your sidebar using the built-in widget.

== Installation ==

1. Upload `Clock-Widget` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Drag and drop clock widget from widget section to any widget area.

== Frequently Asked Questions ==

= How do I display the testimonials in the sidebar? =

Just drag and drop clock widget in widget area.
